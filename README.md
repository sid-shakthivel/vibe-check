# build-day-durham

You are a team of software engineers.
I need you to build the following product:
Imagine a papers please esque game based for banks KYC process.
It should be a web application using ThreeJS (it needs to be 3D)
I do not mind which front end framework you use.
However please use javascript.
The game works as follows:
The player is presented with a 3D environment of a bank
The player interacts with another 3D avatar (the bank manager)
The player has to submit KYC documents to the bank manager and get approved
If there are problems with the KYC documents, the bank manager will reject the player and the player will have to try again
If the KYC is approved, the player wins the game    

The KYC documents needed are:
- Passport
- Proof of address
- Selfie

The application needs to verify the validity of the documents and check if they are genuine - use some api endpoint - if not possible let me know and we can try and use ollama with local models (for security reasons)
